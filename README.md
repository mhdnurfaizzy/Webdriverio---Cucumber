# Webdriverio---Cucumber

**Getting started**

- npm install
- npm init wdio .


**cucumber json | html reporter**

- npm install wdio-cucumberjs-json-reporter --save-dev
- npm install multiple-cucumber-html-reporter --save-dev

===================================================

run with command : `npm run test`
